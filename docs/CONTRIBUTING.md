# Contributing Guide

## Development Setup

### Prerequisites
- Node.js 14+
- MongoDB 4.4+
- Redis (optional)
- Git

### Code Style
- TypeScript strict mode
- ESLint configuration
- Prettier formatting
- Conventional commits

### Branch Strategy
```
main
├── develop
│   ├── feature/xyz
│   ├── bugfix/xyz
│   └── enhancement/xyz
└── hotfix/xyz
```

### Pull Request Process
1. Create feature branch
2. Write tests
3. Update documentation
4. Submit PR
5. Code review
6. Merge
